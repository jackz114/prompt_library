# Prompt Library Browser Extension

这个项目包含两部分：一个简单的 Python 本地服务器和一个 Chrome 浏览器插件。通过它们，你可以直接在浏览器中访问、复制和编辑本地文件夹中的提示词文件。

## 1. 准备工作

确保你的电脑上安装了 Python。

1.  打开终端（CMD 或 PowerShell），进入到项目目录 `e:\projects_all\prompt_library`。
2.  安装依赖（仅需执行一次）：
    ```bash
    pip install -r server/requirements.txt
    ```

## 2. 启动本地服务

这个服务负责读取你本地文件夹中的提示词文件，并提供给浏览器插件使用。你需要保持服务在后台运行。

我们提供了三种启动方式，任选其一即可：

### 方式一：双击启动（推荐）
直接双击项目目录下的 **`启动服务.bat`** 文件。
*   **特点**：会打开一个黑色窗口，显示运行日志。
*   **停止**：直接关闭该窗口即可。

### 方式二：静默启动（无窗口）
直接双击项目目录下的 **`后台启动.vbs`** 文件。
*   **特点**：完全在后台运行，没有窗口干扰。
*   **停止**：需要去任务管理器结束 `python.exe` 进程。
*   **进阶**：可以将此文件的快捷方式放入系统的“启动”文件夹（`shell:startup`），实现开机自启。

### 方式三：命令行启动（高级）
如果你需要自定义参数（例如指定其他文件夹），可以使用命令行：
```bash
python server/main.py
```
*   默认加载项目根目录下的 `prompts` 文件夹。
*   如需指定其他文件夹：`python server/main.py --dir "C:\Users\YourName\Desktop\prompt"`

---

## 3. 安装浏览器插件

1.  打开 Chrome 浏览器（或 Edge）。
2.  在地址栏输入 `chrome://extensions` 并回车。
3.  开启右上角的 **"开发者模式" (Developer mode)**。
4.  点击左上角的 **"加载已解压的扩展程序" (Load unpacked)**。
5.  选择本项目中的 `extension` 文件夹。

## 4. 使用方法

1.  点击浏览器工具栏上的插件图标（如果没有看到，点击拼图图标将其固定）。
2.  你会看到本地文件夹中的所有文件列表。
3.  **单击文件**：查看详情，可以复制内容或编辑保存。
4.  **点击复制**：直接复制内容到剪贴板。
5.  **点击 + 号**：创建新的提示词文件。

## 功能特性

*   **实时同步**：直接操作本地文件，你在本地编辑文件，插件里立即生效；在插件里编辑，本地文件也会更新。
*   **搜索过滤**：快速查找提示词。
*   **编辑功能**：支持在浏览器中直接修改提示词内容。
*   **中文支持**：完美支持中文文件名和内容。
